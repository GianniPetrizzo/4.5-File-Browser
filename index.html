Browser File Handling Observations

  • Image files (PNG/JPG/GIF): The browser displays images directly in a new tab or window, with built-in zoom and navigation controls.
  • Text-based files (HTML, JSON, TXT): HTML files are rendered as web pages, while JSON and plain text files are displayed in a readable format. The browser's built-in text viewer makes these files easily accessible.
  • PDF files: The browser opens up a pdf document using a built in viewer. This viewer has the ability to zoom, rotate, search text, and draw.
